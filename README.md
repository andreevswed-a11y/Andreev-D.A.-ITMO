## О проекте

Репозиторий содержит геоданные и карты для магистерской диссертации, посвящённой разработке метода внедрения водно-зеленой инфраструктуры для адаптации городской среды к условиям изменения климата



## Тема и цель

**Тема:**
ИССЛЕДОВАНИЕ И РАЗРАБОТКА РЕКОМЕНДАЦИЙ ПО АДАПТАЦИИ ВОДНО-ЗЕЛЕНОЙ ИНФРАСТРУКТУРЫ ДЛЯ УПРАВЛЕНИЯ ЛИВНЕВЫМИ СТОКАМИВ УСЛОВИЯХ КЛИМАТИЧЕСКИХ ИЗМЕНЕНИЙ



**Цель исследования:**  
Разработать научно обоснованные рекомендации по адаптации системы управления ливневыми стоками с помощью водно-зеленой инфраструктуры в условиях климатических изменений на примере города Санкт-Петербурга.
